🏋️‍♂️ Fitness & Study Routine Webpage

This project is a simple HTML webpage designed to promote a balanced lifestyle by combining fitness, nutrition, and daily productivity.
It provides a structured diet plan with categorized food items such as proteins, fiber, fats, and carbohydrates to help achieve daily nutritional goals.

The webpage also includes different workout routines suitable for beginners, intermediate, and advanced levels, making it flexible for various fitness stages.
A weekly workout split is presented to ensure consistency and proper muscle group training.

In addition, a daily routine is included to guide users in managing their time effectively between gym, studies, and skill development.
The routine emphasizes discipline, early waking, regular exercise, and consistent learning habits.

The main goal of this project is to create awareness about maintaining a healthy lifestyle while also focusing on academic and technical growth.
It encourages users to track their meals, achieve protein intake goals, and maintain proper sleep.

This project is built using basic HTML and inline CSS, making it beginner-friendly and easy to understand.
It serves as a foundation for future improvements such as adding advanced styling, responsiveness, and interactive features.
